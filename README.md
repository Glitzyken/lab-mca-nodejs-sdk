# Lab: MyCover.ai Node.js SDK

A laboratory Node.js SDK project for MyCover.ai

## Testing locally

Run:

```bash
npm install
```

then...

```bash
npm run build
```

then...

```bash
npm link
```

To continue, clone [this repo](https://github.com/Glitzyken/test-mca-nodejs-sdk) and follow through with the README.
